# IPL 2025 Exploratory Data Analysis

## Project Overview
This project analyzes IPL 2025 match data using Python and performs Exploratory Data Analysis (EDA).

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Data Cleaning
- Missing Value Analysis
- Team-wise Performance
- Top Run Scorers
- Top Wicket Takers
- Win Percentage Analysis
- Data Visualization
